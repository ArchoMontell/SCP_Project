Про проєкт SCP

Фонд SCP — захищена система обліку для небезпечних, аномальних та інших виняткових об'єктів. Проєкт призначений для відстеження камер зберігання, предметів, подій доступу, графіків техобслуговування та операцій безпеки.

Основні функції

  • Керування камерами зберігання та відстеження місткості
  • Класифікація об'єктів (безпечні / небезпечні / аномальні)
  • Планування прибирання та техобслуговування
  • Контроль доступу та реєстрація подій безпеки
  • Звіти та базова аналітика
