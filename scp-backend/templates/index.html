Welcome to SCP Project

The SCP project is a secure containment management system for dangerous, anomalous or otherwise special objects. This prototype demonstrates core ideas: storage units (cameras), object inventory, access control, user roles and event logging.

Core Capabilities

  • Manage containment units (cameras)
  • Track stored objects and their storage requirements
  • User registration & role-based access
  • Event logging for access, maintenance, and incidents
